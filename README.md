# 《情緒修復師：微光城市》

## 遊戲概述
賽道：公益賽道  
核心理念：探討情緒的價值——負面情緒並非純粹的敵人，過度消除負面情緒反而會喪失完整的人性。  
基調與風格：暗黑心理探索、沉浸式共情、非線性碎片敘事。

## 核心玩法與世界觀設計
### 雙層世界架構 (Dual World)
<b>表世界 (微光城市)</b>：碎片化探索，尋找 NPC 遺落的物品或線索（記憶錨點），與NPC對話。  
<b>裏世界 (內心世界)</b>：極端具象化的情緒地獄，NPC背後的故事。  

建議玩家先與NPC對話，發現心理問題，再開始探索城市，蒐集對應的遺落物。透過解鎖物品，線索及降低抗拒值後，才能進入該NPC的內心世界，了解NPC背後的故事，有助於更好地開導NPC。

### 核心收集系統：情緒詞典 (Emotion Dictionary)
在每次探索後解鎖條目。  
包含：情緒名稱、感官描述、NPC 最後告白、玩家註記，以及提供現實心理學建議的 【公益小卡】。

## 後設敘事與安全空間：潛意識酒館 (Subconscious Tavern)
設計者化身：老闆是一個眼袋很深、疲憊但溫柔的年輕人。<br>
情報交換：老闆根據「醫療報告」提供感性的背後故事和遺落線索。<br>
心理緩衝：進場前準備、結束後療癒的中繼站。
