技術認證 PoC：微光城市

這份原型 (Proof of Concept) 證明了我們無需沉重的 3D 引擎，僅用純 Web 前端技術即可實現極具震撼力的視覺效果，完美符合「僅需繳交一個網頁連結」的比賽規則。

1. 表世界：2.5D CSS + 色彩修復

技術驗證：利用純 CSS transform 實現 Isometric 45度視角，結合 filter 實現無縫色彩過渡。成本極低，效能極高。

2. 裏世界：WebGL Shader + UI 故障特效

技術驗證：底層運行 GLSL Fragment Shader (模擬憂鬱深海)，上層疊加純 CSS 製作的 DDLC 式 Glitch 故障文字。
